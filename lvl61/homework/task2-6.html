JavaScript-ს ძირითადად ვიყენებთ კლიენტური და სერვერული მხარეების გასაუმჯობესებლად.

კლიენტურ მხარეზე, JavaScript-ის გამოყენებით ვმუშაობთ ინტერაქტიულობასთან, როგორიცაა ფორმების ვალიდაცია, დინამიური შინაარსის რენდერინგი, მენიუების გამოჩენა და სხვა.

სერვერული მხარეს JavaScript-ის გამოყენებით ვქმნით სერვერებს, რომელზეც აპლიკაციები მუშაობენ, მაგალითად, Node.js-ის გამოყენებით.

MERN Stack არის ვებ აპლიკაციების შესაქმნელად გამოყენებული ტექნოლოგიების ნაკრები:

1. MongoDB - NoSQL მონაცემთა ბაზა, რომელიც JSON ფორმატში ინახავს მონაცემებს.

2. Express.js - Web framework, რომელიც მუშაობს Node.js-ზე და ეხმარება HTTP მოთხოვნების მართვაში.

3. React.js - JavaScript ბიბლიოთეკა, რომელიც შექმნის ინტერფეისს კომპონენტების ფორმით და ეხმარება UI-ს რენდერინგში.

4. Node.js - JavaScript გარემო, რომელიც მუშაობს სერვერზე და ახორციელებს JavaScript კოდს.

JavaScript-ის კოდი შეიძლება დაწერო სხვადასხვა ადგილებში:

1. Inline JavaScript (ატრიბუტებში): JavaScript შეიძლება ჩაეწეროს HTML ატრიბუტში, მაგალითად, <button onclick="alert('Hello')>Click Me</button>

2. Script თეგში (HTML ფაილში): JavaScript კოდი შეიძლება იყოს HTML დოკუმენტში <script> ტეგში, მაგალითად: <script>alert('Hello')</script>

3. External JavaScript ფაილებში: JavaScript კოდი შეიძლება იყოს ცალკე .js ფაილში და ჩატვირთოს HTML-ში <script src="script.js"></script>

როდესაც ღილაკზე დააკლიკებთ, alert box გამოვა, რომელიც გამოგიტანთ თქვენს ასაკს.

ის გამოყენებს JavaScript ფუნქციას, რომელიც გამოხატავს alert-ს, სადაც მითითებულია თქვენი ასაკი.

როდესაც სურათს დააკლიკებთ, მისი src ატრიბუტი შეიცვლება, და ეჩვენება სხვა სურათი.

JavaScript-ის ფუნქცია შეცვლის სურათის src ატრიბუტის მნიშვნელობას.
